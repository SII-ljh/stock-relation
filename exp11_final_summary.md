# 实验11: 最终模型与总结

## 改进历程总结

| 阶段 | NMI | 提升 | 关键改进 |
|------|:---:|:---:|----------|
| 基准 (exp7) | 0.7478 | — | hl=252, pw=0.3, nc=10 |
| Exp8 | 0.8801 | +17.7% | nc=10→42, pw→0.5 |
| Exp9 | 0.8889 | +18.9% | pw→1.0 |
| **Exp10** | **0.8995** | **+20.3%** | pw=0.7, nc=35, TopK=4 |

## 核心改进发现

### 1. 聚类数调整 (nc: 10 → 35)
- 原方案固定10个聚类，但实际有42个行业
- 调整到35后NMI显著提升
- **原因**: 更准确的聚类数能更好匹配真实行业结构

### 2. 先验权重优化 (pw: 0.3 → 0.7)
- 行业先验权重从0.3提升到0.7
- 行业先验信息（同行业相关=1，不同行业=0）比数据更可靠
- **原因**: A股行业结构是股票关系的最强信号
- **注意**: 1.0（纯先验）反而略差，说明适度结合数据有价值

### 3. TopK调整 (k: 5 → 4)
- 每个节点保留4个最强关系
- **原因**: 更稀疏的网络能更好捕捉核心行业结构

### 4. 投资组合验证
- 自适应最佳组合 Sharpe=0.8055
- 显著优于纯先验(0.5930)和等权基准

## 最终最优参数

```python
{
    'half_life': 252,      # 半衰期1年
    'prior_weight': 0.7,   # 先验权重70%
    'n_clusters': 35,      # 聚类数35
    'topk': 4,            # 每节点4个邻居
}
```

**最终NMI: 0.8995** (vs 基准0.7478, **提升20.3%**)

## 关键洞察

1. **行业结构是A股关系建模的核心**: 先验信息比数据驱动的相关性更可靠
2. **聚类数应该接近真实类别数**: 从10调整到35是关键改进之一
3. **适度的先验权重(0.7)优于完全信任数据(0)或完全信任先验(1)**
4. **更稀疏的网络(TopK=4)能更好捕捉核心结构**
5. **自适应协方差估计显著提升投资组合表现**: Sharpe 0.8055 vs 0.5930
