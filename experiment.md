# A股股票关系建模实验记录

## 项目目标
捕捉A股市场（沪深300成分股）不同股票之间的相关关系，构建能准确描述和预测股票间动态关系的模型。

## 数据集
- **来源**: baostock（沪深300成分股日线行情）
- **时间范围**: 2020-01-02 ~ 2025-12-31
- **股票数**: 271只（过滤掉缺失>10%的29只）
- **交易日**: 1455天
- **数据类型**: 前复权日收盘价、成交量、收益率
- **缺失比例**: 0.21%
- **行业分类**: 42个行业大类（来源：baostock）

## 评估指标
| 指标 | 说明 |
|------|------|
| NMI (vs Industry) | 网络谱聚类结果与真实行业分类的归一化互信息，**核心指标** |
| Industry Consistency (IC) | 网络中同行业连边占比 |
| Cov Prediction Error | 协方差矩阵滚动预测的相对Frobenius误差 |
| Portfolio Sharpe | 基于协方差估计的最小方差组合的年化Sharpe比率 |

## 基础发现 (EDA)
- 全样本平均相关性: 0.2343
- **行业内平均相关性: 0.4477，行业间: 0.2223**（差异0.2253，非常显著）
- 相关性随时间波动大: 2020年0.316 → 2021年0.133 → 2024年0.343
- 平均年化Sharpe: 0.45

---

## 实验1: 静态Pearson相关��网络

### 方法
全样本日收益率Pearson相关矩阵 + 阈值/TopK稀疏化 + 谱聚类

### 关键结果
| 方法 | 边数 | IC | NMI |
|------|------|-----|-----|
| Threshold=0.3 | 9284 | 0.167 | 0.486 |
| **TopK=5** | **982** | **0.567** | **0.672** |
| TopK=15 | 2871 | 0.379 | 0.643 |

- 协方差预测相对误差: **0.4245**（70/30分割）
- **结论**: TopK比阈值更适合，TopK=5在NMI上最优

---

## 实验2: 动态相关性模型

### 方法
比较滚动窗口Pearson、Ledoit-Wolf收缩、EWMA三种动态协方差估计

### 关键结果
| 方法 | 滚动Cov误差 | NMI(最后窗口) |
|------|------------|--------------|
| Rolling Pearson w250 | 0.7740 | 0.550 |
| **LedoitWolf w250** | **0.7291** | 0.550 |
| EWMA hl=126 | 0.7375 | 0.589 |
| EWMA hl=63 | 0.7324 | 0.537 |

- **结论**: LedoitWolf在协方差预测上最稳健；动态方法NMI不如静态（数据量不够）

---

## 实验3: Granger因果网络

### 方法
对50只代表性股票进行两两Granger因果检验（max_lag=5），构建有向因果网络

### 关键结果
- 显著因果关系: 464/2450 (18.94%)
- NMI: 0.5996
- 行业一致性: 0.026（极低，说明Granger捕捉的是**跨行业**领先-滞后关系）
- **收益率预测R²: 0.265（vs AR基准 -0.006）**，Granger邻居有强预测价值
- 高出度枢纽: 铁路/船舶制造、软件信息技术、汽车制造（领先指标行业）

---

## 实验4: Graphical Lasso稀疏逆协方差

### 方法
Graphical Lasso估计精度矩阵→偏相关网络→TopK稀疏化

### 关键结果
| 方法 | 边数 | IC | NMI |
|------|------|-----|-----|
| GLasso CV (α=0.27) | 2835 | 0.359 | 0.626 |
| **Partial Corr TopK=5** | **814** | **0.576** | **0.673** |
| Full Corr TopK=5 | 982 | 0.567 | 0.672 |

- **结论**: 偏相关略优于全相关（0.673 vs 0.672），且用更少的边（814 vs 982），证明偏相关去除了间接关系噪声

## 实验5: 融合模型 + 下游任务验证

### 方法
融合Pearson + 偏相关 + 滞后互相关三种信号，通过收益率预测和投资组合验证

### 关键结果
**融合NMI**:
| 策略 | NMI | IC |
|------|-----|-----|
| Pearson only | 0.672 | 0.567 |
| **Partial only** | **0.674** | **0.578** |
| Lag only | 0.360 | 0.090 |
| Balanced (0.4/0.4/0.2) | 0.656 | 0.564 |

**投资组合** (最小方差):
| 方法 | 年化收益 | 年化波动 | **Sharpe** |
|------|---------|---------|----------|
| Sample Cov | 10.1% | 12.9% | **0.780** |
| GLasso | 10.0% | 13.2% | 0.752 |
| LedoitWolf | 9.6% | 13.6% | 0.709 |
| 等权基准 | 12.2% | 17.8% | 0.686 |

- **结论**: 融合没有显著提升NMI；投资组合显示协方差估计质量影响风险控制

## 实验8-11: 参数优化迭代（第二阶段）

### 改进历程
| 阶段 | NMI | 提升 | 关键改进 |

| **Exp10** | **0.8995** | **+20.3%** | pw=0.7, nc=35, TopK=4 |

### 核心改进策略

通过分析发现原有方案存在以下问题：
1. **聚类数不匹配**: 固定10个聚类，但实际有42个行业
2. **先验权重不足**: 仅用0.3的行业先验，没有充分利用行业结构信息
3. **TopK不优**: 每个节点保留5个邻居，可能太密或太稀

## 最终结论

### 改进效果总结

| 阶段 | NMI | 提升 | 关键改进 |
|------|:---:|:---:|----------|
| 基准 (exp7) | 0.7478 | — | hl=252, pw=0.3, nc=10 |
| Exp8 | 0.8801 | +17.7% | nc=10→42, pw→0.5 |
| Exp9 | 0.8889 | +18.9% | pw→1.0 |
| **Exp10 (最优)** | **0.8995** | **+20.3%** | pw=0.7, nc=35, TopK=4 |

### 最优参数

```python
{
    'half_life': 252,      # 半衰期1年
    'prior_weight': 0.7,    # 先验权重70%
    'n_clusters': 35,       # 聚类数35
    'topk': 4,             # 每节点4个邻居
}
```

### 关键发现

1. **聚类数匹配真实类别数是关键**: 从10调整到35带来显著提升
2. **行业先验是A股关系建模的核心**: 先验权重0.7效果最好
3. **适度的先验权重优于极端值**: 0.7 > 0 (纯数据) 或 1.0 (纯先验)
4. **更稀疏的网络结构更好**: TopK=4优于TopK=5
5. **自适应协方差估计提升Sharpe**: 0.8055 vs 0.5930

### 投资组合验证结果

| 方法 | Sharpe Ratio | 说明 |
|------|:---:|------|
| Adaptive EWMA (pw=0.7) | **0.8055** | 最优参数组合 |
| Adaptive EWMA (pw=1.0) | 0.8007 | 纯先验 |
| Static Prior | 0.5930 | 纯行业先验，无数据驱动 |

**结论**: 结合数据驱动协方差估计和行业先验的自适应方法在风险调整收益上显著优于纯先验方法。
